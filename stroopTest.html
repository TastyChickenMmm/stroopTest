A THING

Instructions:

This experiment will consist of 28 rounds, where each round contains 3 steps.

In the first step, you will be shown a colored word, where the font name does not match the name of the color.

e.g.

RED

In the second step, you will be instructed to match either the word or the color for the third step.

e.g.

Select the choice with the matching COLOR.

In the third step, you will be presented with 4 answer choices. Use your keyboard keys (1, 2, 3, or 4) to select the choice which matches either the font name or the color given in Step 1.

e.g.

RED 1

BLUE 2

YELLOW 3

PURPLE 4

In the above example, the correct answer would be YELLOW, because YELLOW and RED are both colored blue, so you should press 3 on your keyboard.

Oh, and also: Speed matters!

Good luck, and press the button below to begin. The experiment should take around 2-3 minutes.

Instructions:

This game will consist of 28 rounds, where each round contains 3 steps.

In the first step, you will be shown a colored word, where the word does not match the color.

e.g.

RED

In the second step, you will be instructed to match either the word or the color.

e.g.

Select the choice with the matching COLOR.

In the third step, you will be presented with 4 answer choices. Use your keyboard (1, 2, 3, or 4) to select the choice which matches either the word or the color given in Step 1.

e.g.

RED 1

BLUE 2

YELLOW 3

PURPLE 4

In the above example, the correct answer would be 3, because YELLOW and RED match in color.

Oh, and also: Speed matters! Try to be fast, but accurate!

Good luck, and press the button below to begin. The game should take around 3 minutes.
